# Pauser

Pauser is an Electron desktop app for Windows that can pause and resume running processes using Sysinternals `pssuspend64.exe`.

It includes:

- A Home page to quickly pause/resume running processes
- Profile-based app groups (pause/resume all apps in a profile)
- A logs dialog for app events and pssuspend output
- Settings to configure the path to `pssuspend64.exe`

## Tech Stack

- Electron
- Vue 3
- Vite
- Tailwind CSS
- Bun (package manager and script runner)

## Requirements

- Windows
- Bun installed
- `pssuspend64.exe` available:
    - Either in this project root (already included in this project)
    - Or available in your system `PATH`
    - Or selected manually in app Settings

## Install

```bash
bun install
```

## Run

Development mode (builds first, opens DevTools):

```bash
bun run dev
```

Normal run (builds first):

```bash
bun start
```

Build renderer only:

```bash
bun run build
```

## Build Standalone Executables

Build unpacked Windows app (direct executable):

```bash
bun run dist:exe
```

Output executable:

- `dist/win-unpacked/Pauser.exe`

Build portable Windows target:

```bash
bun run dist:win
```

Typical output artifacts in `dist/` include:

- `win-unpacked/` (contains `Pauser.exe`)
- portable packaging artifacts generated by electron-builder

## Project Structure

- `electron/` - Electron main and preload scripts
- `src/` - Vue frontend source
- `dist/` - Built frontend and packaging outputs
- `pssuspend64.exe` - Sysinternals suspend tool resource

## Data Storage

Pauser stores user data in Electron `userData`:

- `profiles.json`
- `settings.json`

## Troubleshooting

- `Access denied. Try running Pauser as Administrator.`
    - Run Pauser as Administrator when targeting protected processes.

- `pssuspend64 not found. Configure its path in Settings.`
    - Open Settings in the app and choose the correct `pssuspend64.exe` path.
